# Calendar
A personal calendar which can summarizes the time spent on each type of task.
Distinguishing features : a simplified calendar that provides time analytics and a simplified gantt chart for key projects that may last a few days.

A few issues will be addressed in the coming updates: 
1. The block in the month view is not auto adjusted to the size of the web page
2. The “Add Event” page doesn’t allow the end time to be set to 00:00, which makes it impossible to create an event that spans multiple days and continues past midnight.
3. When adding an event by clicking the time axis in the week view, the default time is always set to the full hour.
